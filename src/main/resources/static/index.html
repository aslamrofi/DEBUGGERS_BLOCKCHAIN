CSBC Digital Identity Management

Simulated W3C Decentralized Identifier (DID) Trust Engine mapped to Oracle 19c.

1. Decentralized Identifier (DID) Generation

Generates an RSA KeyPair. Stored Public Key on Oracle Ledger; Hands Private Key to User Wallet.

2. Cryptographic Identity Verification Portal

Simulates an external verifier verifying the validity of user claims via the public ledger anchor.
